**Установка**. После успешной установки произойдет запуск
<br/>
`chmod +x ./install.sh`
<br/>
`chmod +x ./launch.sh`
<br/>
`./install.sh`
<br/>
**Запуск**
<br/>
`./launch.sh`
<br/>
После запуска в настройках нужно указать свое имя пакета приложения, с которым будет осуществляться взаимодействие
